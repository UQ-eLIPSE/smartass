%%BEGIN QUESTION
If $y=#<v1.y#>$, find $y'$. \\

%%END QUESTION
%%BEGIN SOLUTION

Let $u=#<v1.u#>$, then $u'=#<v1.du#>$.\\
Let $v=#<v1.v#>$, then $v'=#<v1.dv#>$.\\ \vspace{1.3mm}
Quotient rule: $y'=\dfrac{u'v-uv'}{v^{2}}$ ,  
so \\ \vspace{1.3mm}
$y'=#<v1.solution#>.$ \\ \vspace{1.3mm}
Hence $y'=#<v1.shortanswer#>$.
